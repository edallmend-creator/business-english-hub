Lokale Speicherung im Browser

Der Business-English-Hub nutzt den lokalen Speicher Ihres Browsers, um Funktionen der Lernanwendung bereitzustellen. Gespeichert werden können insbesondere Login-Status, Fortschritts-Code, Lernfortschritt, Wiederholungsdaten, Einstellungen, Audio-/Akzent-Einstellungen und selbst gewählte Lernanker.

Diese Daten bleiben grundsätzlich auf Ihrem Gerät. Sie können sie löschen, indem Sie die Website-Daten im Browser entfernen oder den Browser-Speicher leeren. Fortschritts-Code und Synchronisierung

Die App kann einen anonymen Fortschritts-Code erzeugen, mit dem Lernstände auf einem anderen Gerät wiederhergestellt werden können. Für diesen Code werden kein Name und keine E-Mail-Adresse benötigt.
